# 📌 CollabBoard Roadmap

This document outlines the planned phases and release versions for CollabBoard.  
We follow **Semantic Versioning** (`MAJOR.MINOR.PATCH`) for all releases.

---

## **Versioning Guidelines**
- **MAJOR** → Breaking changes (e.g., `v2.0.0`)
- **MINOR** → New features without breaking existing ones (e.g., `v1.1.0`)
- **PATCH** → Bug fixes or minor improvements (e.g., `v1.0.1`)

---

## **Phase 1 → v1.0.0 (Initial Launch)**
🎯 **Target:** Deliver the core functionality for real-time collaboration.

**Features:**
- User Authentication (JWT-based)
- Role-Based Access Control (Admin, Member, Guest)
- Real-time Board Creation (WebSockets)
- Invite Users to Boards
- Basic Task Cards (Create, Edit, Move)

---
