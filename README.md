# 📌 CollabBoard

CollabBoard is a real-time collaboration tool for teams to create, organize, and track work visually.

---

## 🚀 Features (Current)
- Real-time boards and task cards
- Invite team members
- Role-based permissions
- Live updates (WebSockets)

## 🛠 Tech Stack
- **Frontend:** React, TailwindCSS, WebSockets
- **Backend:** Node.js, Express, MongoDB
- **Auth:** JWT
- **Hosting:** Vercel (Frontend), Render/Heroku (Backend)
